# FarmCalc

FarmCalc 是一个面向农机场景的静态单页应用，用于计算设备的每小时/每年运营成本、投资回报率和折旧计划。

## 核心内容

- 运营成本计算器：根据购置价、油耗、维护、保险、仓储和人工等参数生成成本结果
- ROI 分析：计算回本周期、投资回报率和现金流表现
- 折旧表：支持直线法、双倍余额递减法、年数总和法
- 图表可视化：展示成本结构和财务结果

## 技术栈

- Svelte 5
- Vite
- Chart.js
- svelte-chartjs

## 运行方式

```bash
npm install
npm run dev
```
